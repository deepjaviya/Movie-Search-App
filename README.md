# 🎬 Movie Search App (React + Bootstrap)

A modern and responsive movie search web application built with React and Bootstrap. This application allows users to search for movies and view detailed information such as poster, IMDb rating, release year, genre, and plot summary using real-time API data.

🔗 **Live Demo:** https://movie-search-app-99.netlify.app

---

## 📌 Overview

The Movie Search App is a React-based front-end application that integrates with the OMDb API to fetch and display movie data dynamically. It provides a clean user interface and fast performance using React components and Bootstrap styling.

This project demonstrates:

* React functional components
* API integration using Fetch API
* State management using React Hooks
* Responsive design using Bootstrap
* Component-based architecture
* Real-time search functionality

---

## 🚀 Features

* 🔍 Search movies by title
* 🎥 Display movie posters
* ⭐ Show IMDb ratings
* 📅 Display release year and genre
* 📝 Show movie plot summary
* ⚡ Fast and dynamic UI updates using React
* 📱 Fully responsive design using Bootstrap

## 📦 Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/movie-search-app.git
```

2. Navigate to the project folder

```bash
cd movie-search-app
```

3. Install dependencies

```bash
npm install
```

4. Add your OMDb API key in the project

Example:

```javascript
const API_KEY = "your_api_key_here";
```

5. Start the development server

```bash
npm run dev
```

6. Open in browser

```
http://localhost:3000
```

---

## 🔑 API Integration

This project uses the OMDb API to fetch movie data.

Example API request:

```
https://www.omdbapi.com/?apikey=YOUR_API_KEY&t=movie_name
```
